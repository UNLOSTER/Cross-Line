项目：Cross-Line

版本：V1.0

环境：VS2017

简介：简洁的益智小游戏

版权：作者仿造，但代码完全是自己写的，仅用作学习、娱乐，欢迎各位fork!

玩法：很简单，用鼠标拖动可移点，使得所有线段各不交叉，则为过关，其中有不同类别的点，大点可移，小点固定，若一线段不与其他线段交叉，则显白色，否则偏棕色，若一可移点所连线段都不与其他线段交叉，则该点显白色，否则偏棕色，固定点始终偏棕色。
